# Setup

Install [NVM]()  
Then install Node JS by running following command

```bash
nvm use
```

Enable [pnpm](https://pnpm.io/)

```
corepack enable
```

Now, install the packages

```
pnpm install
```

Then create .env file by running following command & add values

```
cp .env.sample .env
```

And

```
pnpm run dev
```
